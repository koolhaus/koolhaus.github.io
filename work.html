BitBox ⟶

Facebook Integrated Bitcoin Wallet. Think Venmo for Bitcoin. Users can sign up with Facebook, deposit and withdraw bitcoin, type in a friends name and send or request bitcoin, view their transaction history and redeem coupons for bitcoin.

  • Built on with Jade/Express/Node/Postgres on top of the Blockchain API
  • Implemented mobile responsive front-end in jade
  • Implemented ACID compliant payment protocol
  • Email Marketing
  • Brand Development
  • First 8,000 Users
  • Initial Investment
  • Founding Team
  • Business Model Development

Quarry ⟶

Package Manager for Paul Graham’s dialect of Lisp, Arc. Central Registry for modular software packages written in arc allowing arc developers to leverage the software ecosystem rather than building everything themselves.

  • Package Manager for arc-language
  • One of the largest projects written in Arc
  • Built Demo Website on the jekyll engine showcasing packages
  • Inspired by npm
